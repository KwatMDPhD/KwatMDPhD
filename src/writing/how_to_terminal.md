## Copy

Select = Op + Select

Without background = Ct + Sh + Cm + C

Plain text = Op + Sh + Cm + C

## Path

Select = Sh + Cm + Double click

Open = Cm + Double click

## Open

Inspector = Cm + I

Characters = Ct + Cm + Sp

## Close

Window with tabs = Sh + Cm + W

Other tabs = Op + Cm + W
